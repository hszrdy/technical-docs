Introduction

Java is a Object-Oriented Programming language that has strongly typed syntax. It is considered to be one of the most well-known programming language.

This documentation serves as an overview of the basic constructs in Java. It will cover:

  • Data Types & Variables
  • Loops & Iterations
  • Methods

There is no prerequisite requirements in reading this document. However, having previous experience in another programming language is good
as this document serves as a quickstart for Java and will not go deep into the details of the language.

JS vs Java

Java and JavaScript are commonly mistaken to be the same programming language but in reality they are completely distinct programming language.
While they are both programming languages, JavaScript is not strongly-typed language unlike Java. Furthermore, JavaScript is dominantly used in web development.

There are more differences and similarities between the two languages that will not be listed here but it is important to distinguish the two languages as seperate.

Data Types and Variables

Data types and variables are important as they serve as the containers of data we use in developing programs.

Creating variables in Java is done through the following syntax:

[data-type] [variable-name] = [value]

Like many other languages, there are multiple data types that Java supports such as:

  • byte
  • boolean
  • char
  • int
  • String
  • float
  • double
  • long
  • short

Example of declaring a variable with a data type of int:

int myIntegerVariable = 42;

There are also non-primitive data types but this document will not cover them.

Loops and Iterations

Loops & Iterations are important constructs in any programming language as it allows programmers to repeat certain lines of code without actually writing them many times.

A example of this is listing even numbers from the range of 1-100. While one may write every even number from that range, it is more convenient to write a loop to list every even number.

There are 3 main looping constructs in Java that a programmer may utilize for iterating. These are:

  • for
  • while
  • do-while

While these loops generally perform the same behavior which is iterating or loop certain lines of code. There might be circumstances where you might prefer to use one over the other.

Generally, for loops are used when there is a set known number of iterations you want to perform such as iterating up to 100 times.

Moving on to the while loop, this loop is commonly used to iterate with unclear number of times, usually iterating until a certain condition is fulfilled such as looping until a boolean is true.

Lastly, the do-while loop. This loop is almost the same as the previous loop construct however this loop is guaranteed to atleast run one time.

Syntax:

for ([variable]; [condition]; [counter])
{
//code
}
while ([condition])
{
//code
}
do
{
//code
} while([condition]);

There is another form of iteration which is called Recursion but this document will not cover it.

Methods

Methods are useful tools in programming. They allow groups of code to be reused in the program. Another common term for methods are functions.
Similar to mathematical functions, methods can take inputs and produce an output but unlike mathematical functions, methods are allowed to have no ouputs or inputs.

The general syntax for a method:

[access-modifier] [method-type] [method-name] ([optional-arguments])
{
//code
//return statement if method-type requires
}

As noticed above, methods can have a type which specifies what type of data they will return or output.
Methods can have primitive data types or non-primitive data types as its return-type/method-type. A special return-type called void specifies the method has no return-type.
